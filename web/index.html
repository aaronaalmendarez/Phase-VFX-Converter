Quick Setup

To upload frames to your Roblox account, you'll need an API key. It takes about 30 seconds.

Open Roblox Creator Dashboard
1
Create API Key Click the blue "Create API Key" button
2
Name it anything e.g. "Phase Converter" — this is just a label
3
Add Assets permission Under "Access Permissions", select Assets → enable Read and Write
4
Set Security (0.0.0.0/0) Under "Security", add 0.0.0.0/0 as an accepted IP so it works from any computer
5
Copy the key Click "Save & Generate Key", then copy it — you won't see it again!

Almost done!

Paste the key you just copied. It never leaves your computer — it goes straight to Roblox.

Your key is stored locally in memory only. It is never saved to disk or sent to any server other than Roblox.
